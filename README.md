# GAS-Scraping-Practice
GASを用いたニュース収集・記録の自動化システム
	
## 概要
毎日手作業で行っていたニュース収集プロセスを完全に自動化。

## システムフロー
1. GASの「日別タイマー」トリガーにより、毎日指定時間に自動実行
2. UrlFetchAppを使い、URLからHTMLソースコード全体を取得
3. 正規表現パターンに基づき、HTMLから記事タイトル、日付、ユニークIDのみを抽出
4. スプレッドシートの既存データを参照し、抽出した記事IDが既に記録されているかを確認
5. 新規記事と判断されたデータのみをスプレッドシートの最終行に一括追記。

## 使用技術
Google Apps Script (GAS)
Google Spreadsheet

## 主な機能
正規表現によるデータ収集
トリガーによる完全自動化
IDによる重複チェック
スプレッドシートへの書き込み

## コードと設定
Webサイトは自作しております。
設定すべき項目：TARGET_URL，sheetName，正規表現パターン
